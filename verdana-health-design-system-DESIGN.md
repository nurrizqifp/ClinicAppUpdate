# Klinik Verdana Design System (Design Tokens & Guide)

## Core Palette
- **Background**: Slate Light (`#F8FAFC` / `bg-slate-50`)
- **Primary Navy**: Slate Dark (`#0F172A` / `bg-slate-900` / `text-slate-900`)
- **Tertiary Sage**: Emerald Medium (`#059669` / `bg-emerald-600` / `text-emerald-600`)
- **Warning Yellow**: Yellow Amber (`#EAB308` / `bg-yellow-500` / `text-yellow-500`)

## Typography
- **Headlines / Titles**: Plus Jakarta Sans (`font-sans` with weight `font-bold` or `font-extrabold`)
- **Body / Labels**: DM Sans (`font-sans` with weight `font-normal` or `font-medium`)
- **Data / Queue Numbers**: Fira Code (`font-mono`)

## Layout Components
- **Borders & Corners**: Rounded 8px (`rounded-lg`)
- **Shadow**: Diffused Shadow (`shadow-sm` or `shadow-md`)
